## 问题分析

在 `ChatView.vue` 中，`getAllWorkflows` 函数被调用时传递了三个参数（1, 50, 'published'），但函数只接受一个参数（对象类型）。这导致了 `TypeError: target must be an object` 错误。

**问题位置：**
`x:\day1\frontend\src\views\ChatView.vue` 第 74 行：
```typescript
const result = await workflowService.getAllWorkflows(1, 50, 'published');
```

**修复方案：**

修改 `ChatView.vue` 中的调用方式，使用对象参数：

```typescript
const result = await workflowService.getAllWorkflows({ 
  page: 1, 
  limit: 50, 
  status: 'published' 
});
```

这样可以确保参数格式正确，符合 `getAllWorkflows` 函数的接口定义。
