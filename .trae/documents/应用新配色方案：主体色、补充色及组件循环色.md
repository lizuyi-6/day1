# 应用全新配色方案

根据您的要求，我将全面更新应用的主题色、补充色以及组件循环色。

## 1. 颜色定义

*   **主体色 (Primary Background)**: `RGB(227, 244, 236)` -> `#E3F4EC`
*   **补充色 (Accent/Primary)**: `RGB(172, 134, 89)` -> `#AC8659`
*   **循环色系列**:
    1.  `#021024` (深蓝黑)
    2.  `#052659` (深蓝)
    3.  `#548383` (灰绿，遵循用户输入)
    4.  `#7DA0CA` (浅灰蓝，修正了原输入中的字母O)
    5.  `#C1E8FF` (极浅蓝)

## 2. 修改计划

### A. 全局样式配置 (`tailwind.config.js`)
*   更新 `colors.primary` 为补充色 `#AC8659`。
*   更新 `colors.background-light` 为主体色 `#E3F4EC`。
*   调整 `colors.sand` 为主体色的深色变体（例如 `#D1E8E0`），以保持视觉和谐。
*   新增 `colors.cycle` 对象，包含上述5个循环色。

### B. 组件样式更新

1.  **ChatView.vue (对话处工作流)**
    *   修改左侧 Agent 列表的图标背景色逻辑。
    *   不再使用统一的 `bg-primary/10`，而是根据 Agent 的索引循环使用 `cycle` 颜色系列。

2.  **WorkflowView.vue (组件/节点)**
    *   更新左侧节点面板的分类颜色。
    *   将硬编码的 Tailwind 颜色类（如 `text-emerald-600`）替换为动态样式，循环应用新的颜色系列。
    *   由于节点类型较多，将按分类或节点顺序循环分配这5种颜色。

3.  **LandingPage.vue (着陆页)**
    *   (可选) 检查 Feature Cards 是否需要应用循环色，如有必要将进行微调。

## 预期效果
应用整体色调将变为清新的淡绿色背景配以稳重的棕褐色强调色。Chat 列表和工作流节点将呈现出从深蓝到浅蓝（及灰绿）的有韵律的色彩变化。
