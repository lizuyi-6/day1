## 页面切换动画实现方案

### 目标
为 Vue 3 项目添加优雅、不过分花哨的页面切换动画

### 当前问题
App.vue 中虽然定义了 `.page-enter-active` 等样式，但**没有实际应用到路由切换上**（缺少 `<transition>` 组件包裹）

### 实现方案

#### 1. 修改 App.vue
- 在 `<RouterView>` 外层包裹 `<transition>` 组件
- 添加 `mode="out-in"` 确保当前页面离开后才显示新页面
- 使用 `v-slot="{ Component, route }"` 获取当前组件和路由信息
- 在 `<component>` 上绑定 `:key="route.path"` 确保切换时触发过渡动画

#### 2. 动画样式（淡入 + 微妙滑动）
```css
.page-enter-active,
.page-leave-active {
  transition: opacity 0.25s cubic-bezier(0.4, 0, 0.2, 1),
              transform 0.25s cubic-bezier(0.4, 0, 0.2, 1);
}

.page-enter-from {
  opacity: 0;
  transform: translateY(8px);
}

.page-leave-to {
  opacity: 0;
  transform: translateY(-4px);
}
```

#### 3. 动画参数说明
- **进入动画**: 250ms，淡入 + 从下方滑入 8px
- **离开动画**: 200ms，淡出 + 向上滑出 4px
- **缓动函数**: `cubic-bezier(0.4, 0, 0.2, 1)` 实现自然流畅的动画

### 预期效果
- 页面切换时会有柔和的淡入淡出效果
- 微微的滑动增加了层次感和流畅度
- 动画时长适中，既提升了用户体验，又不会让人觉得花哨或拖沓

### 实施步骤
1. 读取当前的 App.vue 文件

3. 保留现有的 `<keep-alive>` 和错误边界逻辑
4. 确保动画样式正确应用到过渡组件
