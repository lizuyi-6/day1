## 问题分析

`executionHistoryService.ts` 使用的 API 端口是 `3000`，但后端运行在 `3001`。

**错误**：`http://localhost:3000/execution-history`
**正确**：`http://localhost:3001/execution-history`

## 修复方案

修改 `frontend/src/services/executionHistoryService.ts` 第 3 行：

```typescript
// 修改前
const API_BASE_URL = import.meta.env.VITE_API_BASE_URL || 'http://localhost:3000'

// 修改后
const API_BASE_URL = import.meta.env.VITE_API_BASE_URL || 'http://localhost:3001'
```

这样执行历史就能正常保存了。
