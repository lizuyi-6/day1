## 问题分析

出现了 `Failed to load agents: target must be an object` 错误，这是由于 `getAllWorkflows` 函数的参数处理有问题。

**错误原因：**
- `getAllWorkflows` 函数接受可选的 `params` 参数
- 当 `params` 为 `undefined` 时，Axios 在构建 URL 参数时会调用 `toFormData` 和 `AxiosURLSearchParams`
- 这会导致 `TypeError: target must be an object` 错误

**修复方案：**

1. 确保 `getAllWorkflows` 函数的 `params` 参数有默认值
2. 在调用 `api.get` 时，确保 `params` 是一个有效的对象

**修改步骤：**

修改 `workflowService.ts` 中的 `getAllWorkflows` 函数，确保参数处理正确：

```typescript
export async function getAllWorkflows(params: {

  status?: string
} = {}): Promise<{ success: boolean; workflows?: any[]; error?: string }> {
  try {
    // 确保 params 是一个有效的对象
    const validParams = params || {}
    const response = await api.get('/workflow', { params: validParams })
    ...
  }
  ...
}
```

这样可以确保 Axios 能够正确地处理参数。
