# 重新设计变量选择方式

## 1. 修改 StartNode
- 移除上游变量相关的显示（因为 StartNode 是起点，没有上游）
- 只保留输入变量列表和默认值设置
- 移除 `sourceVariableName` 相关的显示

## 2. 重新设计变量选择方式
- 为每个输入变量添加两种模式：
  - **自定义模式**：直接输入变量名
  - **上游模式**：从下拉列表中选择上游变量
- 添加模式切换按钮（自定义/上游）

## 3. 更新 WorkflowInspector
- 在输入变量配置中，为每个变量添加模式切换
- 自定义模式：显示文本输入框
- 上游模式：显示上游变量下拉列表
- 两种模式都可以设置默认值

## 4. 更新 Variable 接口
- 添加 `mode` 字段：'custom' | 'upstream'
- `sourceVariableName` 只在 upstream 模式下使用
