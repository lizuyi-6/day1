## 创建新的示例工作流

### 目标
删除现有的简单示例工作流，创建一个功能完善的示例工作流，展示系统的多种节点类型和功能。

### 实施步骤

1. **删除现有示例工作流代码**
   - 删除 WorkflowView.vue 第 936-986 行的简单示例

2. **创建新的示例工作流**
   创建一个完整的示例，包含：
   - Start 节点：工作流入口
   - Variable 节点：定义输入变量
   - Knowledge 节点：查询知识库
   - LLM 节点：大模型处理
   - Condition 节点：条件判断
   - HTTP 节点：调用外部 API
   - Email 节点：发送邮件通知
   - End 节点：工作流结束
   
   展示流程：
   ```
   Start → Variable → Knowledge → LLM → Condition (判断结果)
                                     ↓ (是)
                                    HTTP → Email → End
                                     ↓ (否)
                                    End
   ```

3. **配置节点数据**
   - 为每个节点设置合理的默认配置
   - 添加适当的输入输出变量
   - 设置节点位置，使布局美观

4. **测试验证**
   - 确保示例工作流可以正常显示
   - 验证节点连接正确
