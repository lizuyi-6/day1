## 多样化页面切换动画实现方案

### 目标
为不同路由配置不同的切换动画效果，避免单一动画的单调感

### 动画类型映射

根据页面特点，配置以下动画风格：

| 路由 | 动画类型 | 效果说明 |
|-----|---------|---------|
| `/` (landing) | `zoom-fade` | 缩放淡入，从 95% 放大到 100% |
| `/dashboard` | `slide-left` | 从右侧滑入 |
| `/workflows` | `slide-right` | 从左侧滑入 |
| `/workflow/:id` | `slide-up` | 从底部升起，详情页风格 |
| `/chat` | `flip` | 3D翻转效果 |
| `/knowledge` | `fade-scale` | 淡入+轻微缩放 |
| `/workshop` | `zoom-fade` | 缩放淡入 |
| `/about` | `slide-left` | 从右滑入 |

### 技术实现步骤

1. **修改 `router/index.ts`**
   - 为每个路由添加 `meta.transition` 字段
   - 指定对应的动画类型

2. **修改 `App.vue`**
   - 添加计算属性 `transitionName`，根据当前路由的 meta 信息返回对应的动画名称
   - 动态绑定 `<transition :name="transitionName">`
   - 添加所有动画类型的 CSS 样式

3. **CSS 动画细节**
   - 每种动画包含 `enter` 和 `leave` 状态
   - 使用 `cubic-bezier` 缓动函数确保流畅自然
   - 时长控制在 250-400ms 之间，既明显又不拖沓

### 预期效果

- **landing → dashboard**: 主页缩放淡入，工作台从右侧滑入，有"进入系统"的仪式感
- **workflows → workflow**: 列表页从左侧滑出，详情页从底部升起，像打开卡片
- **chat**: 3D翻转增加科技感和趣味性
- **knowledge**: 温和的淡入缩放，适合知识库的学习氛围

这样每个页面都有独特的过渡个性，整体体验更加丰富多样，但又不会过度花哨。是否确认实施？
