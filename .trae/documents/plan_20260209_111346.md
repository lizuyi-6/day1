# 节点连接时变量自动映射实施计划

## 1. 修改 WorkflowView.vue 的 onConnect 事件
- 当连接两个节点时，自动将上游节点的输出变量映射到下游节点的输入变量
- 如果上游节点有多个输出变量，下游节点有多个输入变量，按顺序映射

## 2. 增强 WorkflowInspector 的输入变量配置
- 在"输入变量"标签页中，为每个输入变量添加一个下拉选择框
- 下拉选择框显示所有可用的上游变量
- 支持选择"使用上游变量"或"固定值"
- 显示变量类型和描述信息

## 3. 修改节点数据结构
- 在节点 data 中添加 `variableMappings` 字段，记录变量映射关系
- 格式：`{ [inputName]: { sourceNodeId: string, sourceVariableName: string } }`

## 4. 实现变量映射的 UI 提示
- 在连接边时，显示变量映射的提示信息
- 在节点上显示已映射的变量数量
