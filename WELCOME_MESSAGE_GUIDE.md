# 欢迎语功能使用指南

欢迎语功能已经完整实现！要使用此功能，请按以下步骤操作：

## 步骤

1. **打开工作流编辑器**
   - 访问 http://localhost:5173/workflows
   - 点击你想编辑的工作流卡片

2. **编辑LLM节点**
   - 在画布上点击 LLM 节点
   - 右侧会显示"检查器"面板

3. **配置欢迎语**
   - 在检查器中，找到"欢迎语（首次对话时显示）"字段
   - 输入你想显示的欢迎消息，例如：
     ```
     你好！我是AI助手，有什么可以帮你的吗？
     ```

4. **保存工作流**
   - 点击右上角的"保存"按钮
   - 点击"发布"按钮以部署工作流

5. **测试欢迎语**
   - 访问 http://localhost:5173/chat
   - 从左侧选择你刚才编辑的助手
   - 你应该会看到欢迎语显示为一条系统消息

## 技术细节

- 欢迎语存储在工作流的 LLM 节点配置中
- 只有在首次选择该工作流时显示（如果没有历史记录）
- 如果有历史记录，会优先加载历史记录而不是欢迎语
- 欢迎语会显示为系统消息（灰色虚线边框）

## 验证

如果你想验证欢迎语是否保存成功：

1. 在工作流编辑器中，选择LLM节点
2. 点击"JSON"标签页
3. 查看是否有 `"welcomeMessage": "你的欢迎语"` 字段

如果看到了这个字段，说明欢迎语已经成功保存！
